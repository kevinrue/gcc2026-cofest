This repository tackles a [CoFest project](https://galaxyproject.org/events/gcc2026/cofest/) proposed by Enis Afgan at the Galaxy Community Conference 2026 (GCC2026).

Copying the short description from the page:

> **AI agent to auto-wrap Bioconda/Bioconductor tools**
> A persistent AI agent set up to auto-wrap any new tools added to Bioconda/Bioconductor or similar repos.

The file `CHANGELOG.md` records the process followed so far.
